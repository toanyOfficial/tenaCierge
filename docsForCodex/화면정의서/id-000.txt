# 화면 ID 000 (로그인)

## 공통 규칙
- 카드 하나 안에서 모든 콘텐츠를 표시한다.
- 휴대전화 또는 관리번호 중 하나만 입력해도 로그인이 가능하다는 안내 문구를 노출한다.
- 입력 폼은 단순하게 유지하고, 로그인 버튼만 제공한다.

## 컴포넌트
1. 제목: "로그인" (필수)
2. 안내 문장: "둘 중 하나만 입력하셔도 됩니다." (필수)
3. 입력 필드 A (child-id 1): 휴대전화 번호 (숫자 전용)
4. 입력 필드 B (child-id 2): 관리번호 (placeholder: "5글자고유코드")
5. 버튼 (child-id 3): "로그인" (전체 폭, primary)
