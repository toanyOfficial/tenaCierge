화면 id : 000
화면한글이름 : 로그인화면
화면영어이름 : login
권한 : 누구나
=======================================
컴포넌트 설명
1. inputbox - phone
- 휴대폰번호 입력
- 사용자입력은 하이픈,대시,숫자만 모두 가능하지만 내부 동작에서는 숫자만 남기는 것으로 정규화
2. inputbox - register no
- 5자리 register_no 입력
- 신규채용 시 랜덤으로 발급. 영대문자숫자 섞어서 6자리 랜덤. 중복불가.
3. btn - login
- 위 둘 중 하나만 입력해도 로그인 가능. 로그인은 쿠키를 통해서만 권한관리. 별다른 보안, 권한, 인증 시스템 없음. 쿠키에는 role, phone, register_no를 다 가지고 있는다. 둘 중 하나만 입력했어도 DB 조회해서 나머지 하나의 값을 채워넣는다. null일 수도 있다.
- role은 크게 4가지. Admin, host, butler, cleaner.
- butler는 cleaner중에서 우수한 cleaner를 butler로 임명하는 것이기 때문에 역할은 비슷하지만 조회권환이 더 넓다.
- host는 butler/cleaner와 아예 다른 성격의 role이다
- admin은 모든 정보를 다 볼 수 있다.
- tier가 블랙인 사람은 '접속 불가능한 회원입니다.' alert 띄운다.
